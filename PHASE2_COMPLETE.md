## 🔧 Технические Детали

### База Данных
- PostgreSQL 15 с TypeORM
- 3 таблицы с индексами и foreign keys
- Enum типы для статусов и ролей
- JSONB поля для гибких настроек

### Кэширование
- Redis 7 для сессий
- Конфигурация через environment

### i18n
- i18next с fs backend
- Hot reload через chokidar
- Поддержка контекстов

### Telegram Bot
- Telegraf 4.x
- Middleware цепочка
- Inline клавиатуры

## 🎯 Ключевые Решения

### 1. Микросервисная Архитектура
Каждый сервис независим и масштабируем.

### 2. Горячая Перезагрузка i18n
Переводы обновляются без рестарта.

### 3. Автоматическая Привязка Пользователя
При первом запуске создаётся пользователь.

### 4. Контекстные Языковые Предпочтения
Разные языки для разных частей приложения.

### 5. UUID для Всех ID
Безопасность и масштабируемость.
