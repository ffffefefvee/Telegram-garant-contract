## 🏗️ Система Репутации

### Расчёт Репутации

```typescript
// Базовые дельты событий
DEAL_COMPLETED: +2
DEAL_CANCELLED: -1
DISPUTE_OPENED: -2
DISPUTE_WON: +5
DISPUTE_LOST: -5
RULE_VIOLATION: -10
VERIFICATION_COMPLETED: +10

// Отзывы (5-звёздочная система)
5 звёзд: +5
4 звезды: +2
3 звезды: 0
2 звезды: -3
1 звезда: -5
```

### Уровни Доверия

| Уровень | Диапазон | Иконка | Преимущества |
|---------|----------|--------|--------------|
| New | 0-20 | ⚪ | Базовые лимиты |
| Beginner | 21-40 | 🟡 | Повышенные лимиты |
| Experienced | 41-60 | 🔵 | Доступ к крупным сделкам |
| Reliable | 61-80 | 🟢 | Приоритетная поддержка |
| Verified | 81-100 | ✅ | Максимальные лимиты, бейдж |

### Пример Расчёта

```
Пользователь: start (0)
├─ Завершённая сделка: +2 → 2
├─ Завершённая сделка: +2 → 4
├─ Отзыв (5 звёзд): +5 → 9
├─ Завершённая сделка: +2 → 11
├─ Отзыв (4 звезды): +2 → 13
├─ Отменённая сделка: -1 → 12
├─ Завершённая сделка: +2 → 14
└─ Верификация: +10 → 24 (Beginner 🟡)
```

## 🔐 Безопасность

### Проверки при Создании Отзыва
- Нельзя оставить отзыв самому себе
- Нельзя оставить отзыв без завершённой сделки
- Нельзя оставить повторный отзыв за ту же сделку
- Только опубликованные отзывы влияют на репутацию

### Модерация Отзывов
- Скрытие отзыва по причине
- Удаление отзыва автором
- Анонимные отзывы (скрытие автора)

## 🎯 Ключевые Решения

### 1. Двусторонняя Система Отзывов
- Покупатель → Продавец
- Продавец → Покупатель

### 2. Детальные Оценки
Помимо общей оценки:
- Communication (общение)
- Quality (качество)
- Speed (скорость)
- Accuracy (точность)

### 3. Голосование за Полезность
- helpfulCount / notHelpfulCount
- Процент полезности
- Сортировка по полезным

### 4. Уровни Доверия
Автоматическое определение уровня по score:
- New → Beginner → Experienced → Reliable → Verified

### 5. История Изменений
Полная история всех изменений репутации с:
- Типом события
- Дельтой scores
- Описанием
